# Pose-Bilder

Hier liegen die Pose-Bilder und das App-Icon.

- Dateiname = Pose-`id` aus `data/poses.json`, Endung `.png` – z. B. `img/cobra.png`.
- **Case-sensitive** auf Linux-Hosts: `Cobra.png` lädt nicht und fällt aufs Strichmännchen zurück.
- Fehlt ein Bild oder lädt es nicht, zeigt flowbend automatisch das animierte SVG-Strichmännchen. Bilder sind also optional.
- `icon.svg` ist das PWA-/Browser-Icon.

Empfohlenes Bildformat: quadratisch bis hochkant, transparenter oder heller Hintergrund,
möglichst einheitlicher Stil (die App zeigt sie in voller Breite bis max. 340 px Höhe).
